Authentication

FlowCaptchaPT uses two authentication methods:

API Key (Public endpoints)

Used for captcha token requests. Pass via header:

X-Api-Key: fc_your_api_key_here

API keys start with fc_ prefix.

Admin Token (Management endpoints)

Used for API key management, settings, logs. Pass via header:

Authorization: Bearer your_admin_token

Default: flowcaptcha-admin-2024. Change via FC_ADMIN_TOKEN env.

Error Handling

Code Meaning
401 Missing or invalid authentication
403 API key disabled, expired, or action not allowed
429 Rate limit exceeded (per API key, per minute)
404 Resource not found (job, key, log)
422 Validation error (invalid request body) 
// Error response format
{
  "detail": "Rate limit exceeded (60/min)"
}
POST /api/captcha

Get a reCAPTCHA Enterprise token for Google Flow. This is the primary endpoint for external services.

Headers

X-Api-Key Required. Your API key

Request Body

Field Type Description
action string VIDEO_GENERATION or IMAGE_GENERATION

Response

Field Type Description
token string|null reCAPTCHA token (null if failed/queued)
success boolean Whether token was extracted
status string|null completed, failed, or pending
error string|null Error message if failed
job_id string|null Job ID if request was queued (poll with GET /api/captcha/jobs/{job_id})
callback_url string|null URL to report token result (only on success)

Example

# Request
curl -X POST http://localhost:8899/api/captcha \
  -H "X-Api-Key: fc_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"action": "VIDEO_GENERATION"}'

# Success Response (200)
{
  "token": "03AFcWeA7z6LId-dQ_VVTL0HU9Za3...",
  "success": true,
  "status": "completed",
  "callback_url": "http://localhost:8899/api/captcha/callback/uuid-here",
  "error": null,
  "job_id": null
}

# Queued Response (200) - service busy
{
  "token": null,
  "success": false,
  "status": "pending",
  "error": "Service busy, job queued",
  "job_id": "ad83b729-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "callback_url": null
}

When job_id is returned, poll GET /api/captcha/jobs/{job_id} until status is completed or failed.

GET /api/captcha/jobs/{job_id}

Poll the status of a queued captcha job. No authentication required.

Response

# Pending
{ "job_id": "uuid", "status": "pending", "token": null, "success": false, "elapsed_ms": 5200 }

# Completed
{ "job_id": "uuid", "status": "completed", "token": "03AFcWeA7z...", "success": true, "elapsed_ms": 12400 }

# Failed / Timeout
{ "job_id": "uuid", "status": "timeout", "error": "Timeout after 120s", "success": false }

Possible statuses: pending, processing, completed, failed, timeout. Jobs expire after 5 minutes.

POST /api/captcha/callback/{log_id}

Report whether the captcha token worked. The callback_url is provided in the token response.

Request Body

Field Type Description
result string success or failed
error string|null Optional error code (e.g. CAPTCHA_FAILED, TOKEN_EXPIRED) 
curl -X POST http://localhost:8899/api/captcha/callback/uuid-here \
  -H "X-Api-Key: fc_your_key" \
  -H "Content-Type: application/json" \
  -d '{"result": "success"}'

# Response
{ "ok": true, "message": "Callback recorded: success" }

Code Examples

Python

import requests

BASE = "http://your-server:8899"
KEY = "fc_your_api_key"

# Get video token
resp = requests.post(
    f"{BASE}/api/captcha",
    json={"action": "VIDEO_GENERATION"},
    headers={"X-Api-Key": KEY},
)
data = resp.json()

if data["success"]:
    token = data["token"]
    print(f"Token: {token[:50]}...")

    # Report callback
    requests.post(
        data["callback_url"],
        json={"result": "success"},
        headers={"X-Api-Key": KEY},
    )

elif data.get("job_id"):
    # Poll for result
    import time
    while True:
        job = requests.get(f"{BASE}/api/captcha/jobs/{data['job_id']}").json()
        if job["status"] in ("completed", "failed", "timeout"):
            print(job)
            break
        time.sleep(3)

JavaScript (Node.js)

const BASE = "http://your-server:8899";
const KEY = "fc_your_api_key";

async function getCaptchaToken(action = "VIDEO_GENERATION") {
  const res = await fetch(`${BASE}/api/captcha`, {
    method: "POST",
    headers: { "X-Api-Key": KEY, "Content-Type": "application/json" },
    body: JSON.stringify({ action }),
  });
  const data = await res.json();

  if (data.success) {
    console.log("Token:", data.token.slice(0, 50) + "...");
    return data.token;
  }

  // If queued, poll for result
  if (data.job_id) {
    while (true) {
      const job = await fetch(`${BASE}/api/captcha/jobs/${data.job_id}`).then(r => r.json());
      if (["completed", "failed", "timeout"].includes(job.status)) return job.token;
      await new Promise(r => setTimeout(r, 3000));
    }
  }
  throw new Error(data.error || "Failed");
}
